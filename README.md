# Digital Cafeteria Management System (DCMS)

A mobile application developed as a Final Year Project to digitalize cafeteria operations in university hostels.

## Features
- Student login and authentication
- Daily meal selection
- Cafeteria menu management
- Order tracking
- Admin dashboard

## Technologies Used
- Flutter
- Firebase
- PHP
- MySQL
- Android Studio

## Purpose
To reduce food wastage and improve cafeteria meal management.
